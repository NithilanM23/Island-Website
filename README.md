# 🏝️ Nithilan's Island Portfolio

Welcome to my interactive, gamified portfolio! Unlike traditional static scrolling websites, this is a fully playable isometric island built from scratch using **React**, **Next.js**, and the **HTML5 Canvas API**. 

Walk around the district, step inside buildings to learn about my projects and experience, and even toggle the time of day to watch the island come alive at night.

## ✨ Features

- **🎮 Gamified Exploration:** Walk around a custom isometric world using WASD, Arrow Keys, or on-screen mobile joysticks.
- **🏗️ Interactive Buildings:** Enter different shops (Projects, Skills, Experience, About) to interact with NPCs and discover my work.
- **🎨 Custom Rendering Engine:** The graphics aren't a pre-made game engine—everything is mathematically projected and drawn to an HTML5 Canvas directly in React.
- **🌓 Dynamic Day/Night Cycle:** Toggle between day and night with custom lighting, glowing windows, and fireflies natively rendered on the canvas.
- **📱 Fully Responsive:** Works flawlessly on desktop and mobile browsers.

## 🛠️ Tech Stack

- **Framework:** Next.js (React)
- **Styling:** Tailwind CSS + Vanilla CSS (for glassmorphism UI)
- **Rendering:** Custom HTML5 Canvas Isometric Engine
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/NithilanM23/Island-Website.git
   cd personal-portfolio-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Explore the Island**
   Open [http://localhost:3000](http://localhost:3000) with your browser to start walking around!

## 🎮 Controls
- **Movement:** `W A S D` or `Arrow Keys` (Virtual Joystick on Mobile)
- **Run:** Hold `Shift`
- **Interact:** Press `E`, `Space`, or `Enter` when near an NPC or building.
- **Drone Warp:** Click the Drone icon in the top right to fast-travel.
